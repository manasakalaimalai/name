Face Recognition & Gender Classification Application

As soon as the user uploads the image, the algorithm convert the image into a grayscale, returns a bounding box that crops the face, convert the image into an eigen image and finally uses to model to make predictions.

Gender App

How the Machine Learning Model Works

Read the Image using OpenCV -> Convert the Image to Grayscale -> Crop the Face Using The Haar Classifier
Normalize the Data to Obtain ROI -> Structure the Images by Resizing -> Flatten the Images to Finish Structuring
Begin Feature Extraction Using PCA: Find Eigen Face -> Find Eigen Image -> Pass Eigen Image to SVM ML Model to Obtain Predictions
